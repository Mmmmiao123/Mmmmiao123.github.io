DAY19 HTML基础

一、web标准

​ 由万维网(www=w3c)指定的网页开发规范

​ 结构标准 — html(决定网页上的内容)

二、html

1.什么是html

​ html是超文本标记语言(类似markdown语法)

​ html是通过不同的标记(标签)来决定网页上的不同内容，学写html就是学html中的标签(标记)

2.html语法

​ 1）html大小写不敏感，html == HTML ==HTml == htML

​ 2） 标签(标记)

​ a.语法

​ 双标签：<标签名 属性名1=值1 属性名2=值2 ……>标签内容</标签名>

​ 单标签：<标签名 属性名1=值1 属性名2=值2 …… /> 或者<标签名 属性名1=值1 属性名2=值2 …… >

​ b.说明

​ 标签名 — 必须是html提供的标签名，不能随便写

​ 属性 — 以’属性名 = 值’的形式成对出现的，多对之间用空格隔开；属性的顺序可以任意，不会影响效果

​ 值必须放在双引号中

​ 标签内容 — 只有双标签才有标签内容；标签内容可以是任何内容(不光只能是文字，还可以是其他一个或多个标签)

二、CSS

1.什么是CSS

​ CSS又叫样式表（层叠样式表），是web标准中的表现标准，专门用来针对网页中内容的布局和样式

​ 现在绝大部分使用的是CSS3

2.怎么写CSS

1）语法

​ 选择器{属性1：属性值1；属性2：属性值2}

2）说明

​ a.选择器 — 选中需要设置样式的标签；在内联样式表中，选择器需要省略

​ b.{} — 固定写法

​ c.属性 — 属性是以‘属性：属性值’的形式成对儿出现，多个属性之间要用分号隔开

​ d.值 — 值如果是数字表示大小，数字后面必须加单位：px(像素)，em(空格数)

​ 颜色值：颜色的英文单词、#十六进制的颜色值、rgb(r，g，b)、rgba(r，g，b，a透明度)

​ rgb颜色：r-red(0-255) g-green(0-255) b-blue(0-255)

​ rgb(255，0，0) == #ff0000 == red

​ rgb(00，0，0) == #000000 == black

​ rgb(255，255，255) == #ffffff == white

​ 透明度 — 0(完全透明)~1(完全不透明)

3.写在哪儿

​ 1)内联样式表：将样式写在标签的style属性中；只作用于一个固定的标签

​ 2)内部样式表：将样式写在style标签中；作用于当前html文件中所有的标签

​ 3)外部样式表：将样式写在一个CSS文件中，在html中通过link标签进行导入；可以作用于所有的html中的所有标签

​ 复用性： 外部>内部>内联

​ 优先级： 内联样式表的优先级不管什么情况都是最高，内部和外部的优先级看顺序，谁后写谁优先级高

4.常见属性

​ color — 设置标签中的字体颜色

​ background-color — 设置标签的背景颜色

​ font-size — 设置字体大小

​ width — 设置标签宽度

​ height — 设置标签高度

5.选择器

1.什么是选择器

​ 通过选择器选中标签

2.选择器的写法

1）元素选择器（标签选择器）

​ 直接将标签名作为选择器，选中当前html中对应的所有标签

​ 例如：a{} — 选中当前html中所有的a标签

2）id选择器

​ （所有标签都有一个id属性，值是自己设置的，但是要保证一个html中id的值唯一）

​ 将标签的id属性的值前面加#作为选择器，选中id属性值是指定值的标签

​ 例如：#a — 选中当前html中id值是a的标签

3）类选择器(class选择器)

​ 将标签的class属性的值前面加. 作为选择器，选中class属性值是指定值的标签

​ 1.一个html中相同的class值可以有多个

​ 2.同一个标签的class属性可以有多个（多个之间用空格隔开）

​ 例如：.a — 选中当前html中class值是a的标签

4）通配符

​ 将*作为选择器，选中当前html中所有的标签

5）群组选择器

​ 将多个选择器，用逗号隔开来作为一个选择器，选中每个独立选择器对应的标签

​ p,a{ } — 选中所有的p标签和a标签

​ div，#a{ } — 选中所有的div标签和id值是a的标签

​ #p1，.c1，.c2{ } — 选中id值是p1，class值是c1和c2的所有的标签

6）后代选择器

​ 将多个选择器用空格隔开来作为一个选择器

​ div p{ } — 选中所有div中的p标签

3.伪类选择器

1.什么是伪类选择器

​ 普通选择器选中的是html中不用的标签；伪类选择器选中的是标签的不同状态

2语法

​ 普通选择器：状态{属性1：属性值1；属性2：属性值2；……}

​ 1）link — 初始状态，对于a标签来说，link对应的状态是a标签中的连接从来没有成功访问过；（一般只在a标签进行设置）

​ 2）visited — 作用于a标签，超链接成功访问过的状态

​ 3）hover — 鼠标悬停在标签上对应的状态（不只针对超链接，其他标签也常用）

​ 4）active — 鼠标按住标签不放对应的状态（主要作用在超链接和按钮）

​ 遵守‘爱恨原则 — LoVeHAte’

4.选择器的权重值

​ 权重越大，优先级越高，但是内联样式表的优先级永远最高

​ 元素选择器的权重：0001 == 1

​ class选择器的权重：0010 == 2

​ id选择器的权重： 0100 == 4

​ 伪类选择器的权重： 0001 == 1

​ 群组选择器的权重：看单个每个选择器的权重

​ 后代选择器的权重：所有选择器的权重之和

DAY21 CSS

一、标准流布局

1.什么是标准流

​ 在没有给标签通过css布局的时候，标签在浏览器中有一套默认的布局规则，这个规则就是标准流布局

2.标准流布局规则

​ 块级标签：一个占一行（不管标签本身的宽度）；设置宽高有效；默认宽度是父标签的宽度，默认高度是内容的高度

​ 例如：p，h1-h6，hr，div，ul，ol……

​ 行内标签：一行可以有多个；默认大小就是内容的大小；设置宽高无效

​ 例如：font，a，label，span

​ 行内块标签：一行可以显示多个；默认大小是内容的大小；设置宽高有效

​ 例如：input，button，img

3.display属性

​ block — 块级标签

​ inline — 行内标签

​ inlineblock — 行内块标签

​ none — 隐藏

二、脱流/脱标

​ 只要标签脱流，标准流的规则全部失；不管什么样的标签在脱离标准流的情况下都是按照以下规则进行布局：一行可以显示多个；默认大小是内容的大小；设置宽高有效

​ 浮动和定位都可以让标签脱流

1、float属性

​ left（左浮动） right（右浮动）

​ 浮动能够让标签脱流

2.浮动原理

​ block — 带玻璃板的石头

​ float — 浮水面的叶子

3.浮动内容环绕

​ 被环绕标签不浮动，环绕的内容标签浮动

4.清除浮动

1）高度塌陷

​ 清除浮动是清楚因为浮动而产生的高度塌陷的问题

​ 当父标签不浮动，并且不设置固定高度，子标签浮动就会产生高度塌陷的问题

2）清除浮动的方法

a.空盒子法

​ 在高度会塌陷的标签的最后，添加一个空的div，并且设置这个空的div的样式的clear属性为both

b.hidden

​ 设置高度会塌陷的标签的样式的overflow属性为hidden

c.万能清楚法

​ 给高度会塌陷的标签的after状态添加样式

​ 再给高度会塌陷的标签添加样式属性zoom值为1

​

三、定位

1.position属性

​ 选择定位的标签的参考对象

​ initial/static — 不定位

​ absolute — 绝对定位，相对第一个position属性不是initial/static父标签进行定位

​ relative — 相对定位，相对标准流位置进行定位（相对于原标签在标准流中的位置进行定位）

​ fixed — 相对浏览器定位

​ sticky — 定位保持网页中最后一个块在最后面（网页滚动的时候在浏览器的最下面，网页不超过一屏在内容最下面）

2.left\right\top\bottom

​ 设置当前标签的左、右、上、下到参考对象的左右上下的距离

3.盒子模型

​ border-left-width：10px

DAY23 JS基础

一、什么是JS

​ JS是JavaScript的缩写，是一门专门并且唯一的用来控制网页内容的脚本语言；是web标准中的行为标准—用来控制网页内容的变化。

二、JS写在什么位置

​ 内联 — 将JS代码写在标签的事件相关属性中，例如：onclick属性

​ 内部 — 将JS代码写在script标签中（既可以放在head标签中，也可以放在body标签中）

​ 外部 — 将JS代码写在js文件中，然后通过script标签导入

三、JS能做什么事情

​ 1）修改标签内容

​ 2）修改标签属性

​ 3）修改标签样式

​ 4）添加内容

四、怎么写JS代码

1.注释

2.标识符

​ 由字母、数字、下划线和$符组成，数字不能开头。（注意：$不轻易使用，在JQ有特殊的意义）

3.关键字

​ for if var while func true false…

4.数据类型

​ 1)类型

​ 类型名首字母都是大写的

​ a. Number（数字）、String（字符串）、Boolean（布尔）、Array（数组）、Object（对象）

​ Number — 包含所有的数字（包括整数和小数），例如：100，0.23，-23，2e5

​ b. String — 单引号或者双引号引起来的字符集，支持转义字符和编码字符

​ c. Boolean — 只有true和false两个值，分别代表真和假

​ d. Array — 相当于python中的列表，用中括号括起来的多个元素。[12,’abc’,true,[1,2,3,4]]
